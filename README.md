# Hotel Dream Rimini Website

Multilingual Next.js 16 website (`it`, `en`, `de`) for Hotel Dream Rimini.

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```

## Environment Variables

Create `.env.local` from `.env.example`:

- `NEXT_PUBLIC_SITE_URL`
- `NEXT_PUBLIC_GSC_VERIFICATION`
- `NEXT_PUBLIC_GTM_ID`
- `NEXT_PUBLIC_GA_ID`
